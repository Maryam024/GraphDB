Cypher Help
Basic Commands
  • MATCH (n) RETURN n LIMIT 5 - Find any 5 nodes
  • MATCH (n:Label) RETURN n - Find nodes with a specific label
  • MATCH (n {prop: 'value'}) RETURN n - Find nodes by property
  • CREATE (:Label {prop: 'value'}) - Create a node
  • MATCH (a),(b) WHERE a.prop = 'x' AND b.prop = 'y' CREATE (a)-[:REL_TYPE]->(b) - Create relationship
  • MATCH (n) WHERE n.prop > 25 RETURN n - Filter with WHERE
  • MATCH (a)-[r]->(b) RETURN a,r,b - Match relationships
  • MATCH (n) DELETE n - Delete nodes (and relationships)
  • MATCH (n) SET n.prop = 'value' RETURN n - Update properties
Constraints
  • CREATE CONSTRAINT ON (n:Label) ASSERT n.property IS UNIQUE - Add uniqueness constraint
  • DROP CONSTRAINT ON (n:Label) ASSERT n.property IS UNIQUE - Remove constraint
Transactions
  • BEGIN - Start a transaction
  • COMMIT - Commit changes to the database
  • ROLLBACK - Revert changes in the current transaction
ACID Properties

This graph database implements ACID properties:

  • Atomicity - Transactions are all-or-nothing
  • Consistency - Database remains valid after transactions
  • Isolation - Concurrent transactions don't interfere
  • Durability - Committed changes persist
